sign

  • sign<T>(payload, expirationMs?, issuer?, audience?): IBasaltTokenSignResult

  • Sign a token with the provided payload, issuer, audience, and key pair.

    Type Parameters

    • T extends object

      The type of the payload.

    Parameters

    • payload: T

      The payload of the token, which contains the claims for the token. (T)

    • expirationMs: number = BasaltTokenExpiry.ONE_HOUR

      The amount of milliseconds after which the token should expire. (default is 1 hour) (BasaltTokenExpiry)

    • issuer: string = 'Basalt-Issuer'

      The issuer of the token. (default is 'Basalt-Issuer')

    • audience: string = 'Basalt-Audience'

      [audience='YourAppName-Audience'] - The intended audience of the token. (default is 'Basalt-Audience')

    Returns IBasaltTokenSignResult

    An object containing the signed token, its UUID, and the public key. (The private key is not returned for security reasons, save only the public key and the UUID of the token.) (IBasaltTokenSignResult)

    Example

    Basic usage:

    const tokenResult = basaltToken.sign(\{ user: 'johndoe' \});
    console.log(tokenResult.token); // The signed token

    Example

    Custom expiration time:

    const tokenResult = basaltToken.sign(\{ user: 'johndoe' \}, 1234); // U can use BasaltTokenExpiry enum [30mn, 1h, 2h, 4h, 5h, 6h, 12h, 1d, 1w]
    console.log(tokenResult.token); // The signed token
